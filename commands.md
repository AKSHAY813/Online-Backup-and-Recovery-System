# Project Commands

This file lists the commands available for this project.

## Development

To run the project in development mode with hot-reloading:

```powershell
# Navigate to the project directory
cd "c:/Users/AKSHAY/Desktop/online-backup-system-design/online-backup-system-design"

# Install dependencies (if not already installed)
npm install

# Run the development server
npm run dev
```

## Build

To build the project for production:

```powershell
npm run build
```

## Preview

To preview the production build locally:

```powershell
npm run preview
```
